# 🌦️ Weather App

A simple and responsive weather application that shows real-time weather details based on city input.

---

## 🚀 Features

* 🌍 Search weather by city name
* 🌡️ Displays temperature
* 💧 Shows humidity
* 🌬️ Displays wind speed
* ❌ Handles invalid city input
* 🎨 Clean and modern UI

---

## 🛠️ Tech Stack

* HTML
* CSS
* JavaScript
* Weather API

---

## 📂 Project Structure

```
Weather-App/
│── index.html
│── style.css
│── script.js
│── images/
```

---

## ⚙️ How to Run

1. Clone the repository
2. Open `index.html` in your browser
